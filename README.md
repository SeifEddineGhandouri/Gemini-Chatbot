# 🤖 Gemini Chatbot

A simple chatbot powered by Google’s Gemini API and built with Flask.

## 🚀 Features

- Ask any question and get a response using Gemini AI.
- Simple and clean Bootstrap interface.
- Local server integration with Flask.
